# Interactive Web Pages with JavaScript

## Overview
A simple project demonstrating **interactive web pages** using JavaScript, focusing on **event handling, DOM manipulation, and form validation**.

## Features
- **Dark/Light Mode:** Toggle page theme with a button.  
- **Counter Game:** Increment a number by clicking a button.  
- **Collapsible FAQ:** Click questions to show/hide answers.  
- **Form Validation:** Sign-up form with custom validation and feedback.

## Technologies
- HTML5, CSS3, JavaScript

## How to Use
1. Open `index.html` in a browser.  
2. Interact with buttons, FAQ, and form to see dynamic behavior.  
3. Test form validation by submitting invalid inputs.
